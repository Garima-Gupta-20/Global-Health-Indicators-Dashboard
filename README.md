# 🌍 Global Health Indicators Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using global health indicator data. The dashboard enables users to explore healthcare infrastructure, population trends, life expectancy, and mortality across different countries and regions.

---

## 🎯 Objectives

- Analyze global health indicators
- Compare countries and regions
- Study population and healthcare trends
- Visualize mortality and life expectancy
- Build an interactive dashboard for decision-making

---

## 📂 Dataset

- health_indicators.csv
- indicator_metadata.csv
- country_latest.csv
